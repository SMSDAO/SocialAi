# Screenshots

This directory contains UI screenshots for the README.

## Required Screenshots

1. **public-app.png** - Public timeline and profile pages
   - Should show the GitHub-inspired dark theme
   - Display profile list or timeline view

2. **admin-dashboard.png** - Admin dashboard with metrics
   - Show the main dashboard with worker health
   - Include system metrics and status indicators

3. **desktop-admin.png** - Windows desktop app
   - Screenshot of the Tauri desktop app
   - Show the embedded admin console

## Adding Screenshots

To add screenshots:
1. Take screenshot of the app running locally
2. Name the file according to the list above
3. Place in this directory
4. Screenshots will automatically appear in the README

## Image Guidelines

- Format: PNG (preferred) or JPG
- Resolution: 1920x1080 or higher
- Show the GitHub dark theme UI
- Include realistic data (if possible)
- Crop to remove unnecessary OS chrome
